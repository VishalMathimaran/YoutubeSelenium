# Automating Youtube using Selenium

This project is recommended for lazy users of youtube.

## Installation

Use the package manager [pip3](https://pip.pypa.io/en/stable/) to install necessary packages.

```bash
pip3 install selenium
```


## Usage

```bash

python3 main.py
```
